TREND 1

Flat 2.0

Apple deciding to abandon the 3D skeumorphic style with iOS 7 (2013) established still running and evolving trend.

Inspired by: Bauhaus,
swiss style and zune player.

Read more about zune player →

Simple vector shapes stand for objectivity
and perform better on smaller screens.

See pocopay project by AKU →

After minimalism
it's time for essentialism.

See Trident Gum packaging project →

TREND 2

Bright Bold
Colors

In recent years we’ve seen a move from muted colours to a more saturated palette.

Windows 8: the boldest,
biggest redesign in Microsoft's history.

Read more about Windows 8 →

Vibrant hues should continue
to be popular picks.

See Arrels by heystudio →

TREND 3

Modu
layout

Modular or card-based layouts have been adopted by some of the biggest brands for their websites and mobile apps.

Modular layout stand for responsive,
adaptability and coherence.

See Pharrell Williams website →

Modularity keeps evolving
in different areas of design.

Check ARA modular phone's website →

TREND 4

Organ
chaos

In contrast to the more excessive,
80's inspired styles, a rebellious, minimalist trend is evolving.

Things Organized Neatly:
The Art of Arranging the Everyday

Check Austin Radcliffe's blog →

Playing with various elements
on a fixed grid.

See Autor Rooms by mamastudio →

TREND 5

Material
design

In 2014 Google created its styleguide about Material Design which specified animation, style, layout, patterns and usabilty.

Using 3D capabilites that allow objects
to have dimensions on X, Y and Z-axis.

Check Google’s Material Design styleguide →

Playing with shadows can create
an impression of space.

Check Allegra project by DHNN →

Don't follow
trends blindly

Know them and play with them wisely.
